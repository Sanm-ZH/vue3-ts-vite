vue3 vite setup typescript
基于vue3 vite 使用typescript vue setup的学习项目
## Project setup
```bash
pnpm install
```

### Compiles and hot-reloads for development
```bash
pnpm run dev
```

### Compiles and minifies for production
```bash
pnpm run build
```

### Lints and fixes files
```bash
pnpm run lint
```
